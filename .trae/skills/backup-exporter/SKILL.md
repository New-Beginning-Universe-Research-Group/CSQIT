# Backup Exporter

## Overview

This skill provides automated backup and export functionality for Lean projects. It packages project files, exports all Lean code to text format, and creates comprehensive documentation.

## Features

1. **Complete Project Backup**: Copies all Core modules and configuration files
2. **Lean Code Export**: Exports all `.lean` files to a single text document
3. **Documentation Generation**: Creates comprehensive project documentation
4. **Timestamped Backups**: Automatically generates date-stamped backup directories

## When to Invoke

- User asks to "backup", "export", or "archive" project files
- User requests to "save", "package", or "copy" the project
- Before major changes or deployments
- When preparing for review or handoff

## Usage Workflow

### Step 1: Create Backup Directory
```powershell
$timestamp = Get-Date -Format 'yyyy-MM-dd_HH-mm-ss'
$backupDir = "CSQIT_10.4.5_FinalBackup_$timestamp"
New-Item -ItemType Directory -Force -Path $backupDir
```

### Step 2: Copy Core Files
```powershell
Copy-Item -Recurse -Force 'Core' "$backupDir\"
Copy-Item -Force 'lakefile.lean' "$backupDir\"
```

### Step 3: Export All Lean Code
```powershell
$files = Get-ChildItem -Path '.' -Filter '*.lean' -Recurse
$output = "CSQIT 10.4.5 - Complete Export`r`n"
$output += "Date: $timestamp`r`n"
$output += "Files: $($files.Count)`r`n`r`n"

foreach ($file in $files | Sort-Object FullName) {
    $relativePath = $file.FullName.Replace($PWD.Path + '\', '')
    $output += "==========================================`r`n"
    $output += "FILE: $relativePath`r`n"
    $output += "==========================================`r`n"
    $output += [System.IO.File]::ReadAllText($file.FullName)
    $output += "`r`n`r`n"
}
[System.IO.File]::WriteAllText("$backupDir\All_Lean_Code_Full.txt", $output)
```

### Step 4: Generate Documentation
Create comprehensive documentation including:
- Project overview
- Module structure
- Axiom system summary
- Key theorems
- Open problems
